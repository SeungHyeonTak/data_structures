원소 삭제 방법
del(li[2]) -> li[2]인 30을 삭제 하고 index를 하나씩 앞당긴다.
* del() -> 특정 인덱스에서 항목을 제거
계산 복잡성 -> O(n-1)
* pop() -> 특정 인덱스에 있는 항목을 삭제 해 돌려준다.
계산 복잡성 -> O(n-1)
* remove() -> 특정 색인이 아닌 처음으로 일치하는 값을 remove 제거함.
계산 복잡성 -> O(n)

※ del인덱스를 기준으로 pop()으로 인해 요소를 제거하고,
반환값이 필요할 경우 인덱스를 사용하여 제거하고,
값이 remove() 기준으로 요소를 삭제할 때 사용함.
